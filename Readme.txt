Following instructions has been tested in Ubuntu 14.04.

1. Download the stable release from http://pyjs.org/Download.html
2. Clone this system of files and folders, place them inside the downloaded pyjs directory
3. Install python 2.7
4. go into the game directory(from terminal)
5. Enter this command ./../bin/pyjsbuild witgame.py
6. To play the game in web browser enter this command firefox output/witgame.html
 
Note: the game is supported in all modern day browsers. So in place of firefox you can use any other browser name.

Enjoy!!!
